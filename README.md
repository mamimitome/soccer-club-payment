# サッカークラブ 月謝管理・決済システム

SES研修中の学習プロジェクトです。
サッカークラブの月謝徴収をオンライン化することを目的に開発しています。

## 開発状況

現在開発中です（2026年4月〜）

- [x] Laravelプロジェクト環境構築
- [x] データベース設計・マイグレーション
- [x] ログイン・認証機能（Laravel Sanctum）
- [x] 管理者ダッシュボード
- [x] 会員ダッシュボード
- [x] Stripe決済機能の実装
- [x] 決済テスト完了
- [x] 会員管理CRUD機能
- [x] 本番環境へのデプロイ

## 使用技術

| 分類           | 技術                    |
| -------------- | ----------------------- |
| バックエンド   | Laravel 12（PHP 8.2）   |
| 認証           | Laravel Sanctum         |
| フロントエンド | Blade / Tailwind CSS v4 |
| データベース   | MySQL                   |
| 決済           | Stripe                  |
| バージョン管理 | Git / GitHub            |

## 機能概要

### 管理者機能

- ログイン・ログアウト
- 会員の追加・編集・削除
- 支払い状況一覧（月別・未払い確認）
- 売上ダッシュボード
- ビジター代行決済

### 会員機能

- ログイン・ログアウト
- 月謝支払い（9,000円 / 月）
- 支払い履歴確認
- マイページ

### ビジター機能

- ログイン・ログアウト
- 都度払い（2,500円 / 回）
- 支払い履歴確認
